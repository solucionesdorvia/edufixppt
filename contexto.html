Contexto de la validación EduFix

Esta página resume el propósito del panel y enlaza la presentación donde se desarrolla el problema, la propuesta y el diseño del producto.

Qué es el panel

El archivo index.html es una visualización interactiva de las respuestas de encuesta: distribuciones, medias y tabla filtrable. Los porcentajes de las tarjetas de hallazgos se calculan a partir de esos mismos datos.

Validación del problema

Las encuestas buscan medir cómo la comunidad educativa percibe el reporte de desperfectos: si se ignora el problema, si no se sabe a quién recurrir o cómo usar el canal oficial, y qué impacto tiene en la experiencia. Esa lectura alimenta la narrativa de validación del deck EduFix.

Datos y reproducibilidad

Los registros se generan con generar_respuestas.py, que escribe respuestas.csv y data.js. Volvé a ejecutar el script después de cambiar parámetros si necesitás otro tamaño de muestra.

La presentación PowerPoint con gráficos y hallazgos (solo lectura del CSV) se genera con generar_presentacion.py → EduFix_Pitch_Venta.pptx (pitch para proyector). Requiere pip install -r requirements.txt.
